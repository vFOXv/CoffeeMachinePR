Используемые технологиии:
Java, Collection, Spring, Thymeleaf, HTML

Приложение моделирует работу кофемашины.
При запуске по URL "/" открывается меню приложения
1 Make coffee   переход в подменю с выбором видов кофе которые можно сделать,
                при переходе ссылке Make, please :)
                Если не хвотает ингридиентов, то автоматически переход на страницу их пополнения.
2 Statistics    На странице указываеться кол-во сделаных видов кофе за сессию и возможно просмотреть рецепт каждого вида кофе.
                Также кол-во потраченных ингридиентов, и кол-во ингридиентов в кофемашине.
                Есть сылка на пополнение ингридиентов(Add resources CoffeeMachine).
3 Service       Пополнени ингридиентов. При добавлении болле чем объем бака, эта часть ингридиетнов теряеться.
